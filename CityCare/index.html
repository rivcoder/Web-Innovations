    How CityCare Works

    Use the Report tab to submit issues.
    Authorities manage them from History.

    Steps to Report:
    1. Upload a clear image.
    2. Select issue type.
    3. Enter or auto-detect location.
    4. Add description (optional).
    5. Submit and track in History.

    Guidelines:
    • Provide accurate location.
    • Avoid duplicate reports.
    • Emergency issues should be reported directly to authorities.
