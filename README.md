🚀 Projects
🏙️ CityCare
Civic-tech platform bridging citizens and local authorities
A community-driven web app for reporting and resolving urban issues — potholes, broken infrastructure, waste management. Citizens submit reports with images and location data, the community upvotes what matters most, and authorities get a structured dashboard to act on it. Gamified leaderboard rewards active contributors.
Key Features: Smart issue reporting with location + image upload · Community upvotes for priority ranking · Live city map · Authority dashboard with lifecycle tracking (Sent → In Progress → Resolved) · Points & credibility system

🌪️ Disaster Whisper
Real-time environmental risk estimator
Enter any city and get a live risk assessment powered by real weather data and an ML model. Uses a hybrid approach — Random Forest predictions combined with rule-based corrections — to output a Risk Index (%) with plain-language interpretation.
Key Features: Live weather data integration · ML-based risk scoring (Random Forest) · Hybrid ML + rule-based system · City search · Real-time updates

🚨 SituAid
Emergency first-response guide
When something goes wrong, panic sets in and time is wasted. SituAid cuts through that — input the patient's condition and get clear, step-by-step guidance on what to do before help arrives. Built for real emergency situations where knowing the right action matters.

⚔️ Chaos Arena
A card-battle browser game built on pure randomness
Each click triggers a card draw for both you and the enemy. Cards are randomly assigned power levels — weak cards give small points, strong cards give bonus points and confetti, villain cards deal heavy penalties. Every match is different. Strategy barely exists. Chaos is the point.
